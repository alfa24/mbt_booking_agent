# План: Задача 03 - Выбор инструментов миграций и доступа к БД

## Цель

Подготовить ADR с обоснованием выбора инструментов для миграций и доступа к БД, а также практическую справку по их использованию.

## Состав работ

### 1. Исследование вариантов миграций

| Вариант | За | Против |
|---------|-----|--------|
| Alembic (SQLAlchemy) | Стандарт, автогенерация, async | Требует знания SQLAlchemy |
| Migrate (Django-style) | Простота | Требует Django ORM |
| Ручные SQL-скрипты | Полный контроль | Высокий риск ошибок |

### 2. Исследование вариантов ORM/доступа к БД

| Вариант | За | Против |
|---------|-----|--------|
| SQLAlchemy 2.0 + asyncpg | Нативный async, Pydantic v2 | Кривая обучения |
| psycopg3 (raw) | Минимальная абстракция | Ручной SQL |
| databases + sqlalchemy-core | Легковесный | Меньше функциональности |

### 3. Создание ADR-003

Содержание:

- Обоснование выбора SQLAlchemy 2.0 + asyncpg

- Trade-offs и риски
