# План: Задача 08 — Актуализация соглашений и документации

## Цель

Обновить проектную документацию на основе реализованного backend.

## Контекст

Backend реализован согласно ADR-002 (FastAPI + SQLAlchemy 2.0 + PostgreSQL). Структура проекта:
- `backend/api/` — роутеры (endpoints)
- `backend/models/` — SQLAlchemy модели
- `backend/schemas/` — Pydantic схемы
- `backend/services/` — бизнес-логика
- `backend/repositories/` — слой доступа к данным
- `alembic/` — миграции

Все операции с проектом, миграциями и тестами выполняются только в Docker.
