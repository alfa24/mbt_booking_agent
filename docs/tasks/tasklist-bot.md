## Задача 01: Базовая структура бота 🚧

### Цель

Создать каркас Telegram-бота с подключением к LLM и базовой обработкой сообщений.

### Состав работ

- [x] Инициализация проекта
- [ ] Подключение aiogram
- [ ] Интеграция с LLM-провайдером
- [ ] Базовый обработчик сообщений

### Артефакты

- `bot/main.py` - точка входа
- `bot/config.py` - конфигурация
- `bot/handlers/message.py` - обработчики
