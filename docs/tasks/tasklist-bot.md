## Задача 01: Базовая структура бота ✅

### Цель

Создать каркас Telegram-бота с подключением к LLM и базовой обработкой сообщений.

### Состав работ

- [x] Инициализация проекта
- [x] Подключение aiogram
- [x] Интеграция с LLM-провайдером
- [x] Базовый обработчик сообщений

### Артефакты

- `bot/main.py` - точка входа
- `bot/config.py` - конфигурация
- `bot/handlers/message.py` - обработчики

## Задача 02: Обработка естественного языка ✅

### Цель

Научить бота понимать намерения пользователя: даты, количество гостей, выбор дома.

### Реализация

- [x] Интеграция LLM с function calling (OpenAI-compatible API)
- [x] Определение tools для бронирования
- [x] Обработка responses с tool calls

### Цель

### Реализация

- [x] Интеграция с Backend API через HTTP client
- [x] Вызов BookingService для операций
