## Задача 01: Настройка проекта frontend ✅

### Реализовано

- Next.js 15 + React 19 проект
- shadcn/ui компоненты
- Tailwind CSS v4 стилизация
- Zustand для state management
- TanStack Query для data fetching

---

## Задача 02: Аутентификация через Telegram ✅

### Реализовано

- Ввод Telegram username
- Сохранение в Zustand store
- Middleware для проверки доступа
- Разделение по ролям (owner/tenant)

---

## Задача 03: Каталог домов и календарь ✅

### Реализовано

- Страница `/tenant/houses` - каталог домов
- Страница `/tenant/houses/[id]` - детали дома
- Календарь занятости
- Кнопка "Забронировать"

## Задача 04: Управление бронированиями ✅
